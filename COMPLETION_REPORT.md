## 📚 Ключевые изменения API

### Пример: функция start()

**Было (v13.7):**
```python
def start(update: Update, context: CallbackContext) -> int:
    update.message.reply_text('Привет!')
    return NAME
```

**Стало (v20.7):**
```python
async def start(update: Update, context: CallbackContext) -> int:
    await update.message.reply_text('Привет!')
    return NAME
```

## 🎓 Технические детали

### Асинхронный цикл событий (asyncio)
- Application использует asyncio.run_forever()
- Все обработчики выполняются в asyncio event loop
- Телеграм polling работает асинхронно

### ConversationHandler состояния
```
START
  ├─> NAME (ввод имени)
  ├─> CODE (проверка кода)
  ├─> PHONE (ввод телефона)
  ├─> CONFIRM_PHONE (подтверждение)
  ├─> PROBLEM_TYPE (выбор типа)
  ├─> SUB_PROBLEM_TYPE (выбор подтипа)
  ├─> ANYDESK (ввод кода AnyDesk)
  ├─> TEXT_DESCRIPTION (описание проблемы)
  └─> END
```
